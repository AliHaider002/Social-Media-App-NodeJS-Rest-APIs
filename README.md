REST Api Social Media App in NodeJs.

Routes = {Auth, User, Post}. 
Controllers = {AuthController, UserController, PostController}.

Login User, 
Create user,

Follow/UnFollow User, 
Get User, 
Create Post, 
Update Post, 
Delete Post,
